📋 Complete Input → Process → Output Flow
# Module Input Process Output / Stored In
1 🔑 Authentication INUsername, Password / OTP PROCValidate credentials, check MFA, verify device OUTSession token → DBsessions, verified_browsers
2 🚗 Unit Management INPlate No., Make, Model, Year, Cost, GPS IMEI PROCAuto-assign boundary rate via BoundaryRule, status management OUTUnit record → DBunits
3 💰 Boundary Collection INUnit, Driver, Date, Actual Amount Collected PROCCalculate shortage/excess vs expected, record status, flag incentive OUTDaily income record → DBboundaries
4 🔩 Maintenance INUnit, Type, Description, Labor Cost, Parts, Mechanic PROCLink parts, update unit status to maintenance, toggle complete/in-progress OUTMaintenance record → DBmaintenance, maintenance_parts
5 🚦 Coding INPlate number last digit / Manual override PROCAuto-detect coding day (M-Sa), flag units, generate system alert OUTCoding status → DBcoding_records, system_alerts
6 👤 Driver Management INName, License, Contact, Type, Status, Documents PROCLink to user account, assign to unit (primary/secondary), track status OUTDriver profile → DBdrivers, users
7 ⚠️ Driver Behavior INIncident type, Accident details, Debt amount PROCRecord incident, calculate debt installments, track incentive eligibility OUTBehavior log → DBdriver_behavior, driver_debts
8 🧾 Office Expenses INCategory, Amount, Date, Description PROCSubmit → Approval workflow → Approved/Rejected OUTExpense record → DBexpenses
9 📡 GPS Tracking INIMEI number request to Tracksolid API PROCAPI call with time-drift fix, parse coordinates, update location OUTLocation data → DBgps_tracking, units.latitude/longitude
10 📊 Dashboard INHTTP request (auto-refresh 30s) PROCAggregate DB queries: boundaries, expenses, maintenance, units, drivers OUTKPI Cards, Charts, Alerts → Real-time JSON API
11 📈 Analytics INDate range filter PROCCompute revenue trends, ROI per unit, expense breakdown, top drivers OUTCharts & Reports → Browser render
12 📦 Archive System INDelete action on any record PROCSoft-delete (set deleted_at), remove from active views OUTArchived record (restorable) → DBdeleted_at column
13 💵 Salary INStaff/Driver, Amount, Period PROCRecord salary, compute monthly totals OUTPayroll records + Monthly Report
14 ⚙️ Boundary Rules INYear range + Rate amount PROCStore rule, auto-apply when unit year is set OUTRate config → DBboundary_rules → auto-sets units.boundary_rate
15 🐙 GitHub CI/CD INGit push to main branch PROCGitHub Actions triggers workflow, SSH deploy to Hostinger OUTLive production update at eurotaxi.hostinger.com
